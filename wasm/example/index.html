SILO WASM Example

Minimal example showing how to write files into the Emscripten filesystem, preprocess them with SILO, save the processed state, load it again, and run SaneQL.

Serve this page over HTTP with Cross-Origin-Opener-Policy: same-origin and Cross-Origin-Embedder-Policy: require-corp, because the current SILO WASM build uses pthreads.

1. Preprocess input files

Upload the files referenced by your preprocessing config, for example preprocessing_config.yaml, database_config.yaml, reference_genomes.json, and an NDJSON input file.

2. Load processed state and query

The JSON file is only this example's archive format; SILO itself saves and loads a directory of processed state files.
